= 耐障害性を設定
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp HCI に Rancher をインストールした後、仮想マシンの配置を最適な復元力に設定する方法について説明します。

次の点に関して耐障害性を考慮してください。

* 負荷分散またはラウンドロビンプロキシを設定すると、管理クラスタの耐障害性が向上します。
* ユーザクラスタの場合は、ユーザクラスタをどのように設定するかに応じて、耐障害性をセットアップできます。
